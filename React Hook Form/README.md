# CARWEB3: React Hook Form - Apresentação

Este projeto foi desenvolvido com o objetivo de apresentar os principais recursos do **React Hook Form**, demonstrando como criar e validar formulários de maneira simples, performática e organizada. 
Feito por Eduarda Gonçalves
## 🚀 Tecnologias

- React
- React Hook Form
- Vite
- JavaScript/TypeScript (conforme o projeto)

## 📦 Instalação

Clone o repositório e instale as dependências:

```bash
npm i
```

## ▶️ Executando o projeto

Para iniciar o servidor de desenvolvimento:

```bash
npm run dev
```

Após executar o comando, acesse a URL exibida no terminal (geralmente `http://localhost:5173`).

## 🏗️ Gerando a build

Para criar a versão de produção:

```bash
npm run build
```

Os arquivos da build serão gerados na pasta `dist`.

## 👀 Visualizando a build

Para visualizar localmente a versão de produção:

```bash
npm run preview
```

Após executar o comando, acesse a URL informada no terminal.
