# Apresentação - React Motion (Framer Motion)

## Tema

Demonstração dos principais recursos de animação utilizando Framer Motion no React.

## Integrante
 Vitor R.

## Requisitos

* Node.js 18+
* npm ou yarn

## Instalação

```bash
npm i
```

## Executando o projeto

```bash
npm run dev
```

O projeto ficará disponível em:

```text
http://localhost:3000
```

## Demonstrações disponíveis

1. whileHover
2. whileTap
3. animate
4. initial + animate
5. AnimatePresence
6. layout
7. whileInView
8. rotate
9. scale

## Objetivo

Apresentar os principais recursos de animação da biblioteca Framer Motion, demonstrando interações, animações de entrada e saída, animações contínuas e animações baseadas em scroll.
